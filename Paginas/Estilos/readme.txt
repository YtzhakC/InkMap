La página de estilos se creó para organizar y presentar los estilos de tatuajes
de cada artista de una forma visual y atractiva. Cada tatuador cuenta con una 
sección dedicada a sus estilos, permitiendo que los usuarios exploren los tipos
de tatuajes que cada artista domina y elijan el estilo que mejor se adapte a lo 
que buscan.

Características:

Estilos Personalizados para Cada Tatuador: Cada tatuador tiene su propia sección
en la que se muestran los estilos en los que se especializa, como realismo, oriental
y tradicional.
Galería de Ejemplos: Para cada estilo, se muestran imágenes de tatuajes realizados 
por el tatuador, de modo que los usuarios puedan ver ejemplos específicos.
Filtro por Estilos: Los usuarios pueden filtrar por estilos de tatuajes para encontrar
rápidamente los artistas que trabajan en el estilo deseado.
Diseño Responsivo: La página está optimizada para dispositivos móviles y de escritorio, 
permitiendo una experiencia visual fluida en cualquier dispositivo.
